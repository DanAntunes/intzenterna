Sobre

Qual o objetivo desse projeto?

O objetivo deste projeto é criar um site de torcedor para a INTZ, uma das maiores organizações de e-sports da América Latina. O site servirá como um almanaque para a torcida, compartilhando histórias felizes e tristes da nossa Magnética. Este projeto também faz parte de um projeto universitário e será usado como portfólio.

Quem está por trás deste projeto?

Este projeto é desenvolvido por um estudante universitário apaixonado por e-sports e pela INTZ . O estudante queria criar um espaço onde outros torcedores se identificassem e tivessem um lugar com bastante conteúdo sobre a nossa amada Organização.

O que torna este projeto único?

Este projeto é único porque é um dos poucos sites de torcedores dedicados a uma organização específica. Ele oferece uma perspectiva única e pessoal sobre a organização e seus jogadores.
